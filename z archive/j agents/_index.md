# Agents — Index

Front door for everything AI-agent related. Each subfolder has an obvious home for one kind of thing.

## Subfolders

- [[Skills]] — one note per skill / SKILL.md I build or collect
- [[Prompts]] — reusable prompts and system-prompt snippets
- [[Tools & MCPs]] — connectors / MCP servers I've set up, their configs and gotchas
- [[Logs]] — notable runs, outputs worth keeping, things that broke

## Conventions

- New skill note? Copy [[Skill Template]] into `Skills/`.
- Tag everything `#agent` plus a sub-tag (`#agent/skill`, `#agent/prompt`, `#agent/mcp`, `#agent/log`).
- Keep notes atomic and link liberally — folders are just the front door, links are the map.
