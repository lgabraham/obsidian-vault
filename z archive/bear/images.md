# images

I think the best conception of the “lead product image” today would be the following image
* First a product class image 
* that is the first image for a given store
* from this set of stores, in order to 
  * Urban Outfitters
  * Lulu and Georgia
  * West Elm
  * Pottery Barn
  * AllModern
  * Joss and Main
  * Birch Lane
  * Wayfair

So for a given product variant, if Lulu and Georgia has product images, we’d select the product image that is earliest in their sequence, i.e. the “first” product class image we see. But if Urban Outfitters has a product image, we’d select this image — the earliest product class image in the sequence that we see.
