# Rounds
Gadient's pro rata at $xm valuation:
* $25m - $375k
* $30m - $750k
* $35m - $1.125m
* $40m - $1.5m
 So if we raise $8m on $40m, then we'd have:
* oHouse for as much as $4m
* Gradient for $1.5m
* Remainder is only $2.5m
  Or if we raise $6m on $30m, then we'd have:
* oHouse for as much as $3m
* Gradient for $.75m
* Remainder is only $2.25m
 Or if we raise $5m on $25m, then we'd have:
* oHouse for as much as $2.5m
* Gradient for $.375m
* Remainder is only $2.125m
 Basically, in the event, we take max oHouse money, we are raising $2-$2.5m (edited)  Not $4m  Of course, we can take 40% so that if raise $6m on $30m, then we'd have:
* oHouse for $2.4m

* Remainder is only $2.85m
