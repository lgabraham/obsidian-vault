To quickly check your reveal status, use the api tool which can be viewed here: https://api.seekers.xyz/seeker/44495

simply replace the X's with the number of your beacon (collectable ID).

Once you see the text/code of the API, look in the last bit of code for  “animation_url” The last part of the file name there will say standard, rare, mythic or unrevealed
